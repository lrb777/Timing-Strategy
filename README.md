# 基于三线反向突破图与蜡烛图形态的A股择时策略（无监督学习优化）

## 策略概述

本策略以日本技术分析中的**三线反向突破图**（Three-Line Break Chart）作为趋势过滤器，以**蜡烛图形态**作为具体入场触发信号，以**转向线**作为出场信号，并引入**无监督学习**对策略参数和市场状态进行优化。

择时标的：**沪深300指数**（以300ETF或股指期货交易）  
数据周期：**日线**  
交易方向：**多头/空仓**（基准为持有/不持有）

理论来源：史蒂夫·尼森《日本蜡烛图技术新解》第六章"三线反向突破图"

---

## 策略框架

```
三线图判断大趋势方向
        ↓
蜡烛图形态寻找具体入场点
        ↓
转向线作为止盈出场信号
        ↓
无监督学习完善策略
```

---

## 各模块说明

### 模块一：三线反向突破图（趋势过滤）

**构图规则**（仅使用收盘价，忽略时间轴）：
- 收盘价创新高 → 画白线（多头）
- 收盘价创新低 → 画黑线（空头）
- 收盘价位于前一根线区间内 → 不画新线
- **转向线条件**：价格必须突破前三根连续同色线的极值（最高点/最低点），才能画出反向转向线

**趋势状态定义**（待确认）：
- 方案A：当前线颜色即代表趋势方向（白=牛市，黑=熊市）
- 方案B：出现三根连续同色线才确认趋势（信号更少但更稳）
- 横盘期（价格不突破区间，不画新线）：维持前状态或空仓

**参数**：
- N 值（反转所需突破的连续线数，默认 N=3，可调整为 2/4/5）

---

### 模块二：蜡烛图形态（入场信号）

仅在三线图确认**牛市状态**时，识别以下多头形态触发买入：

| 形态 | 类型 | 量化条件（待细化） |
|------|------|------------------|
| 锤子线 | 单根 | 下影线 ≥ 实体×2，上影线 ≤ 实体×0.1 |
| 启明星 | 三根 | 长黑+小实体+长白，第三根收盘超过第一根中点 |
| 多头吞噬 | 双根 | 白色实体完全吞噬前一黑色实体 |
| 刺穿线 | 双根 | 白线收盘超过前一黑线实体中点 |
| 高浪线 | 单根 | 上下影线均极长，实体极小 |

**执行规则**：收盘识别形态，次日开盘入场（A股T+1限制）

---

### 模块三：转向线出场（止盈信号）

- **出场条件**：收盘价跌破前三根白线的最低点，三线图出现**黑色转向线** → 次日开盘平仓
- **备选止损**：是否设置独立止损线（防止转向线出现前回撤过大，待讨论）

---

### 模块四：无监督学习优化（待第一至三模块验证后实施）

在基础策略回测完成后，引入无监督学习：

1. **市场状态聚类**：以波动率、趋势强度、三线图连续线段长度等为特征，用 K-Means 或 DBSCAN 将历史市场分为若干状态，在不同状态下自适应选择 N 值
2. **蜡烛形态有效性聚类**：对历史形态的后续收益分布聚类，筛选在三线图多头环境中真正有效的形态子集
3. **目标**：提升策略在不同市场环境下的稳健性，减少参数过拟合
