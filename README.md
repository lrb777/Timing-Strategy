# 基于三线反向突破图与蜡烛图形态的A股择时策略（无监督学习优化）

## 策略概述

以**三线反向突破图**（Three-Line Break Chart）作为趋势过滤器，以**蜡烛图形态**作为入场触发信号，以**转向线**作为出场信号，并引入**无监督学习**对策略参数和市场状态进行优化。

- 择时标的：沪深300指数（`000300`）
- 数据周期：日线
- 交易方向：多头 / 空仓
- 数据来源：akshare
- 理论来源：史蒂夫·尼森《日本蜡烛图技术新解》第六章

---

## 策略框架

```
三线图判断大趋势方向
        ↓
蜡烛图形态寻找具体入场点
        ↓
转向线作为止盈出场信号
        ↓
无监督学习完善策略
```

### 模块一：三线反向突破图（趋势过滤）

仅使用收盘价构图，忽略时间轴：
- 收盘价创新高 → 画白线；收盘价创新低 → 画黑线；位于区间内 → 不画线
- **转向线**：价格突破前 N 根连续同色线的极值才能画出反向线（默认 N=3）
- 趋势状态：当前线为白线 = 牛市，黑线 = 熊市

### 模块二：蜡烛图形态（入场信号）

仅在三线图确认牛市时，识别以下多头形态触发买入（收盘识别，次日开盘入场）：

| 形态 | 量化条件 |
|------|---------|
| 锤子线 | 下影线 ≥ 实体×2，上影线 ≤ 实体×0.1 |
| 启明星 | 长黑＋小实体＋长白，第三根收盘超过第一根中点 |
| 多头吞噬 | 白色实体完全吞噬前一黑色实体 |
| 刺穿线 | 白线收盘超过前一黑线实体中点 |

### 模块三：转向线出场

收盘价跌破前三根白线最低点 → 出现黑色转向线 → 次日开盘平仓

### 模块四：无监督学习优化

基础策略验证后，对历史市场状态聚类（波动率、趋势强度、三线图线段特征），在不同状态下自适应选择 N 值与形态组合。

## 项目结构

```
Timing-Strategy/
├── README.md
├── requirements.txt
├── .gitignore
├── config/
│   └── config.yaml          # 所有策略参数（N值、形态开关、回测参数、ML参数）
├── data/
│   ├── raw/                 # 原始下载数据（不入库）
│   └── processed/           # 处理后数据（不入库）
├── src/
│   ├── data/                # 数据获取模块
│   ├── indicators/          # 三线图构建 + 蜡烛图形态识别
│   ├── strategy/            # 主策略逻辑（信号生成 + 仓位管理）
│   ├── backtest/            # 回测引擎
│   └── ml/                  # 无监督学习模块
├── notebooks/               # 探索性分析
├── tests/                   # 单元测试
└── results/
    └── figures/             # 输出图表（不入库）
```
